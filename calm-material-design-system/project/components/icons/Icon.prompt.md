Renders a Material Symbols Rounded glyph — the Calm Material icon set. Use for every icon; never hand-draw SVGs or use emoji.

```jsx
<Icon name="photo_camera" />
<Icon name="favorite" fill color="var(--color-primary)" />
```

- Default 24px, weight 400, outline style. `fill` for active/selected states.
- Tint `var(--color-primary)` for active/brand, default `on-surface` for neutral.
